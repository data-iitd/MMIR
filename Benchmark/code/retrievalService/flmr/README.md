# FLMR Retrieval Service

## Aim
This directory contains the retrieval server for PreFLMR using ColBERT. It serves the ColBERT indices for caption, joint, and image retrieval.

## Usage

### `preflmr_colbert_server_all3.py`
Starts the PreFLMR ColBERT server.
```bash
python preflmr_colbert_server_all3.py
```
(This script includes a `main` block that runs uvicorn on port 5052).
